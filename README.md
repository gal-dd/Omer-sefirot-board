# Omer Sefirot Board

אתר אינטראקטיבי להצגת **בניין הכוונות לפי 7 הספירות בספירת העומר**.

המערכת מאפשרת מעבר בין שתי מערכות:
- **מערכת דחסדים**
- **מערכת דמוחין**

וכן בחירה של כל אחד מ־49 ימי העומר, עם הצגה חזותית של המילוי המצטבר בלוח המרכזי.

---

## תכונות עיקריות

- תצוגת **49 ימי העומר**
- לוח מרכזי של **7×7**
- הצגת **מילוי מצטבר** עד היום שנבחר
- תמיכה ב־2 מערכות ספירות:
  - חסדים
  - מוחין
- **מצב יום / מצב חושך**
- ממשק RTL בעברית
- עיצוב מודרני ונקי
- מוכן לפריסה ב־**GitHub Pages**

## איך זה עובד

האתר מציג את 49 ימי העומר כצירוף של:
- **ספירת השבוע**
- **ספירת היום**

לדוגמה:
- יום 1: חסד דחסד
- יום 2: חסד דגבורה
- יום 8: גבורה דחסד

בעת בחירה של יום מסוים, הלוח מציג את **כל מה שהתמלא עד אותו יום**, ולא רק את הפעולה של אותו יום בלבד.
